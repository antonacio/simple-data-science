# Data Science Examples

Useful examples for common Data Science use cases:
1. Classification

3. Clustering
4. Dimensionality Reduction

    - UMAP

## Setup

### Pre Commit

```bash
pip install -r requirements.txt
pre-commit install
```

or

```bash
make install-pre-commit
```
